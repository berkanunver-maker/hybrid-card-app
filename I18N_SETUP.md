# İ18n Kurulum Komutları

## Gerekli paketleri yükleyin:
```bash
npm install i18next react-i18next expo-localization
```

## Veya yarn ile:
```bash
yarn add i18next react-i18next expo-localization
```

## Paket Açıklamaları:
- **i18next**: Çok dilli destek için core kütüphane
- **react-i18next**: React Native için i18next integration
- **expo-localization**: Cihaz dilini algılamak için
